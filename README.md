# Musion Post

**An AI-powered label manager** that acts as a proactive partner guiding artists through the entire release lifecycle—from finished track to sustained growth.

## What it does

Musion Post is not a passive assistant. It's an AI manager that:

- **Analyzes data** — streaming performance, audience demographics, platform engagement
- **Makes strategic decisions** — optimal release timing, platform prioritization, audience targeting
- **Executes promotional tasks** — Spotify editorial submissions, playlist pitching, press outreach, social scheduling, ad campaigns
- **Manages your release calendar** — every event AI-scheduled and tracked
- **Guides the full lifecycle** — from Concept → Production → Pre-Release → Launch → Post-Release → Growth

Artists approve high-level strategy. The AI handles 80% of execution.

## Tech stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Recharts** — analytics visualizations
- **Lucide React** — icons

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

| Page | Description |
|------|-------------|
| **Dashboard** | Real-time overview — KPIs, active release, AI recommendations, upcoming calendar events |
| **Releases** | Full release pipeline from concept to growth, with AI scoring and task tracking |
| **AI Strategy** | Data-driven recommendations with confidence scores, impact/effort ratings, and one-click AI execution |
| **Calendar** | Visual promotional calendar — every event AI-scheduled and color-coded by type |
| **Analytics** | Stream performance, audience demographics, platform breakdown, and follower growth charts |
| **Tasks** | AI-generated and AI-executed promotional tasks across the entire release lifecycle |
| **Settings** | Artist profile, AI automation preferences, and plan management |
